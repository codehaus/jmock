Anonymous CVS Access

This project's CVS repository can be checked out through anonymous (pserver) CVS with the following instruction set. When prompted for a password for anonymous, simply press the Enter key.

cvs -d:pserver:anonymous@cvs.jmock.codehaus.org:/scm/jmock login

cvs -d:pserver:anonymous@cvs.jmock.codehaus.org:/scm/jmock checkout module

Where module is jmock2 for jMock 2 and jmock for jMock 1.

Updates from within the module's directory do not need the -d parameter.

Building from Source

Running Ant in the root of the jMock source tree will perform a clean build and ZIP packages of the JAR files and Javadoc documentation in the build subdirectory. We use Ant version 1.7.
